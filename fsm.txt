┌────────────────────────────────────────────────────────────────┐
│ TIMEOUT GLOBAL (5 segundos):                                    │
│                                                                │
│ Quando to_active = 1:                                          │
│ - to_cnt incrementa a cada ciclo                              │
│ - Quando to_cnt ≥ TIMEOUT_VAL (5.000.000 µs):                │
│   └─ digitos_value ← {20{0xE}} (erro)                         │
│   └─ digitos_valid ← 1 (pulso)                                │
│   └─ fsm ← ST_CLR                                             │
│                                                                │
│ to_active é ATIVADO em ST_IDLE quando key_pulse = 1           │
│ to_active é DESATIVADO quando digitos_valid = 1               │
└────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────┐
│ BUFFER DE DÍGITOS (20 × 4 bits = 80 bits):                     │
│                                                                │
│ digits[19] = mais antigo (primeiro digitado)                   │
│ digits[0]  = mais recente (último digitado)                    │
│                                                                │
│ Quando confir: todo buffer é enviado (com 0xF em posições      │
│ não preenchidas)                                               │
│                                                                │
│ Códigos de saída:                                              │
│ - 0x0-0x9: Dígitos (0-9)                                       │
│ - 0xA: * (asterisco/confirmação)                               │
│ - 0xB: # (hash/reset)                                          │
│ - 0xE: Erro (timeout)                                          │
│ - 0xF: Inválido (posição não preenchida)                       │
└────────────────────────────────────────────────────────────────┘
